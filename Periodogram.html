Periodogram
¶

朱琨
2019.12.13

$~~~~~~$给定一个平稳过程$\{x_t\}$的观测数据$x_0,x_1,...,x_N$时，当$N$足够大时，如何估计功率谱密度？估计功率谱密度最早的方法为周期图方法(Periodogram)。由于序列$\{x_t\}$的离散傅里叶变换$X(m)$具有周期性，因而功率谱也具有周期性，所以称为功率谱方法。

1.伪代码：¶

Input：输入观测数据$x_1,x_2,...x_n$ .
Step1:计算 $$X(m)=FFT\{x_1,x_2,...,x_n\},m=0,1,2..,N$$

Step2:计算 $$P(\frac{m\pi}{N+1})=\frac{\mid X(m)\mid^2}{N}$$

Output:$P(\frac{m\pi}{N+1})$在$[0,\pi]$上的柱状图及其包络

import numpy as np
import matplotlib.pyplot as plt
from scipy.fftpack import fft
import math
pi=math.pi
#生成N个正弦信号
def generate_sin(N):
    '''
    :param 
            N: number of sample
    :return: 
            xn:the A series of signals#随机信号
    '''
    # sample rate采样率
    fs = 1000
    # N 个样本
    t = np.linspace(0, 0.3, N)
    # 振幅
    A = np.array([2, 8]).reshape(-1, 1)
    # 频率
    f = np.array([150, 140]).reshape(-1, 1)
    # 产生信号
    xn = (A * np.sin(2 * np.pi * f * t)).sum(axis=0) + 5 * np.random.randn(*t.shape)
    return xn
'''
Step1:generate the a random signals
'''
# number of sample
N = 301
# 产生随机信号
x = generate_sin(N)
#绘制原始信号图
plt.plot(x)
#标题名
plt.title("Original Signal Plot")
#x轴名称
plt.xlabel("Frenquency")
plt.ylabel("Amplitude")
plt.show()
'''
Step2:Discrete Fourier Transform to x
'''
#对离散x做离散傅里叶变换
X = fft(x)
# Magnitude(amplitude)
mX = np.abs(X)
# phase
pX = np.angle(X)
'''
 Step3:Calculate the |X(m)|^2/N
'''
P = (1/N)*np.square(mX)
'''
Step4: Plot the graph
'''
#归一化
norm_P =[]
#最大值
max_P = np.max(P)
#最小值
min_P = np.min(P)
#对P(t)做归一化
for t in P:
    t = (t-min_P)/(max_P-min_P)
    norm_P.append(t)
#绘制周期图
plt.plot(norm_P)
#标题
plt.title("Periodogram")
#x轴名称
plt.xlabel("Frenquency")
#y轴名称
plt.ylabel("power spectrum")
plt.show()

$~~~~~~$然而，它并不是真正的周期图。当$N\rightarrow\infty$时，偏差不趋于0.所得的周期图随$N$的变化而变化，这种现象称为随机起伏。
